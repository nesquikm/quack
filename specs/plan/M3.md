# Implementation Plan

## Milestone Order

M1 (Bootstrap) shipped pre-toolkit; M2 (multi-tenant auth + admin MCP + `server_status` + Docker) shipped and archived. M3 is the memory plane: Neo4j wiring, the four primitive memory MCP tools, the extraction loop, the cleanup sweep, and client-side hook scripts that close the end-to-end loop.

## M3: Memory plane (Neo4j + extraction + memory MCP tools + hooks) {#M3}

**Goal:** Full memory pipeline lit end-to-end — Claude Code hooks POST to `/ingest`, extraction runs against Neo4j, memory-plane MCP tools return structured DTOs scoped per project, the cleanup sweeper drains deleted projects' graph data, and `server_status` reports real queue + extraction counters.

**Prerequisites:** M2 shipped (green gate, auth + admin MCP + Docker working; M2's `compose.yml` + `auth.sqlite` schema in place).
